# 2026 KBO Exhibition Games Dashboard

2026 KBO 시범경기 스탯 리더보드 — 정적 JSON 기반 독립 배포 프로젝트

## Features

| Page | Route | Description |
|------|-------|-------------|
| **Leaderboard** | `/exhibition` | 타자/투수 시범경기 주요 지표 리더보드 |
| **Pitcher Profiler** | `/exhibition/pitcher-profiler` | 투수 구종/구위 프로파일 분석 |
| **Prospects** | `/exhibition/prospects` | 신인/유망주 시범경기 성적 |
| **Velocity & Contact** | `/exhibition/velocity-contact` | 직구 구속 vs 컨택트 분석 |
| **Player Detail** | `/exhibition/:pcode` | 개별 선수 상세 (일별 성적, 피칭, 상대전적) |

### Filters

- **구단별 필터링** — 10개 KBO 구단 드롭다운
- **외국인/신인 필터** — 해당 페이지별 토글
- **최소 타석/이닝 필터** — 슬라이더/입력
- **정렬** — 모든 컬럼 클릭 정렬 (클라이언트 사이드)

## Tech Stack

- **React 18** + **TypeScript 5**
- **Vite 6** (build & dev server)
- **Tailwind CSS 3** (utility-first styling)
- **React Router 6** (SPA routing)
- **Static JSON** (`/data/*.json`) — no backend required

## Getting Started

```bash
# Install dependencies
npm install

# Dev server
npm run dev

# Production build
npm run build

# Preview build
npm run preview
```

## Data Generation

JSON 데이터는 별도 export 스크립트로 생성합니다:

```bash
# API 서버가 실행 중인 상태에서
python ops/scripts/export-exhibition-json.py --season 2026
```

Output: `public/data/` 디렉토리에 정적 JSON 파일 생성

## Deployment (Vercel)

```bash
npx vercel deploy
```

`vercel.json`에 SPA rewrite 설정 포함.
